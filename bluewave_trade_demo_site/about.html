About this project

A simple trading-themed website focused on blue design, static components, and easy-to-copy layout.

What this site is

This website is a front-end prototype for a trading platform. It includes a landing page, asset catalog, per-asset detail pages with charts, and mock registration and login screens. Everything is implemented as static HTML and CSS.

You can use it as a starting point for coursework, design reviews, or future development once a backend is connected. The color system is centered around blue tones to create a focused, trustworthy feel.

Design principles

Consistent spacing, reusable components, and clear hierarchy. Cards, buttons, and forms share the same rounded, blue-oriented style.

No real trading

All financial values and actions are mock data. This prevents accidental use as a real product and keeps the focus on layout and UX.

Easy to extend

Because every page is static, you can plug in any framework or backend later without rewriting the whole interface.
